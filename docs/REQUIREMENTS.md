# Требования к системе записи PsyBooking

## Цель проекта
Создать современную систему онлайн-записи к психологу с уведомлениями и удобным интерфейсом.

## Целевая аудитория
- Основные пользователи: Клиенты психолога (записывающиеся на сеансы)
- Администратор: Психолог (управление расписанием, просмотр записей)

## Основные функции

### Для клиентов:
- Просмотр доступного времени в календаре
- Запись на сеанс с выбором даты и времени
- Указание контактных данных (имя, телефон, Telegram)
- Краткое описание запроса (опционально)
- Подтверждение записи
- Получение напоминаний о сеансе

### Для психолога:
- Мгновенные уведомления о новых записях
- Просмотр всех записей в календаре
- Управление расписанием (блокировка времени)
- Контактная информация клиентов
- Отмена/перенос записей
- Настройки уведомлений

## Технические требования

### Уведомления:
- Telegram уведомления (опционально)
- SMS напоминания клиентам за 24 часа до сеанса

### Интерфейс:
- Адаптивный дизайн (мобильные устройства)
- Интуитивно понятный интерфейс
- Быстрая загрузка (< 3 секунд)

### Безопасность:
- Защита персональных данных
- HTTPS соединение
- Валидация всех форм

## MVP (Минимальный продукт)
1. Форма записи с календарем
2. Простая админ-панель для просмотра записей

## Будущие функции
- Интеграция с Google Calendar
- Система оплаты онлайн
- CRM интеграция
- Мобильное приложение
